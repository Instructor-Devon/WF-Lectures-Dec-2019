## Introducing jQuery (broswer-side JS)

We can use JavaScript to provide dynamic behavior to our HTML pages!  All HTML elements are scriptable, and configurable using JavaScript.  This can be done using "vanilla" javascript, but jQuery provides a handy set of features to do this easier.

### New Concepts:
- document loading
- callback functions
- anonymous functions
- event listening/handling

### jQuery Concepts:
- "selectors"
- getters vs. setters
- API Documentation
